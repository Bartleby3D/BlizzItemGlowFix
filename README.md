# BlizzItemGlowFix

BlizzItemGlowFix is a World of Warcraft addon that improves Blizzard item buttons by adding configurable item quality borders, item levels, stack counts, and status icons across multiple item-related frames.

## Features

* Configurable item quality borders and glow styles.
* Item level text on supported item buttons.
* Custom stack count text with configurable font, size, position, alignment, and color.
* Junk item icon.
* Upgrade icon for eligible items.
* Uncollected transmog icon.
* Quest item icon.
* Missing enchant icon.
* Per-module enable/disable controls.

## Supported frames

* Bags
* Character frame
* Inspect frame
* Character bank
* Warband bank
* Guild bank
* Merchant frame
* Mail frame
* Trade frame
* Loot frame
* Quest rewards
* Profession journal
* Crafting orders

Some modules are marked as experimental.

## Usage

Open the settings window with either command:

```text
/bigf
/blizzitemglowfix
```

The settings panel contains sections for:

* General font settings
* Stack count text
* Border style and border behavior
* Item level text
* Status icons
* Module toggles

## Notes

BlizzItemGlowFix only changes visual presentation on Blizzard item buttons. It does not modify items, inventory data, rewards, or gameplay logic.
